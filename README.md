# Генетический алгоритм с видообразованием

## Описание проекта

Проект посвящён исследованию и реализации **генетических алгоритмов с видообразованием** для задач **дискретной и непрерывной оптимизации**.

Основной акцент сделан на:
- разделение популяции на виды;
- динамические роли (например, хищник–жертва);
- исследование выхода из локальных минимумов;
- подбор параметров (ручной и автоматический).

Проект используется в рамках исследовательской курсовой работы.

## `continuous_ga_core/`

Реализация генетического алгоритма для **непрерывной оптимизации**.

### Состав

- `pipeline.py`  
  Реализация классов генетических алгоритмов с видообразованием.

- `population.py`  
  Представление популяции и особей.

- `evaluation.py`  
  Функции для непрерывной оптимизации.

- `selection.py`  
  Операторы селекции.  

- `crossover.py`  
  Операторы скрещивания.

- `mutation.py`  
  Операторы мутации.

- `clusterization.py`  
  Алгоритмы кластеризации особей в виды.

- `interaction.py`  
  Логика взаимодействия между видами.

- `termination.py`  
  Критерии остановки алгоритма.

- `logger.py`  
  Логирование метрик оптимизации.

- `visualization.py`  
  Визуализация процесса оптимизации и динамики популяции.

---

## `discrete_ga_core/`

Реализация генетического алгоритма для **дискретной оптимизации** (в разработке).

## `experiments/`

Каталог с экспериментами и исследовательскими ноутбуками.

### Jupyter notebooks

- `base_experiments.ipynb`  
  Базовые эксперименты с генетическим алгоритмом.

- `comparison_of_continuous_optimization.ipynb`  
  Сравнение различных стратегий непрерывной оптимизации.

- `pushing_out_of_the_local_minimum.ipynb`  
  Исследование механизмов выхода из локальных минимумов.

- `selection_of_parameters_hand.ipynb`  
  Ручной подбор параметров генетических алгоритмов.

- `selection_of_parameters_hand_2.ipynb`  
  Дополнительные эксперименты по ручной настройке параметров.

- `selection_of_parameters_auto.ipynb`  
  Автоматизированный подбор параметров.

---

## `utils/`

Вспомогательные утилиты общего назначения.

- `timeit.py`  
  Измерение времени выполнения алгоритмов и отдельных этапов.

- `visualization.py`  
  Общие функции визуализации, используемые в экспериментах.
